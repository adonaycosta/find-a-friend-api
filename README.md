#  Find a Friend API

A Node.js API built to simulate a pet adoption system, focusing on business rules, clean architecture, and automated testing.

---

## 🎯 About the Project

This project was developed as a **study challenge**, with the goal of improving backend development skills, especially:

* Building REST APIs
* Structuring applications using layered architecture
* Applying business rules
* Writing automated tests

---

## 🧠 Business Rules

* Pet registration
* Filtering pets by city, age, and characteristics
* Data validation
* Simulated adoption flow

---

## 🛠️ Technologies

* Node.js
* TypeScript
* Prisma
* Docker
* Vite
* Zod

---

## 🧪 Tests

This project includes automated tests focused on business logic, ensuring:

* Reliability of core features
* Validation of key scenarios
* Safer code evolution

Run tests with:

```bash
npm run test
```

---

## 🧱 Architecture

The project follows a structured approach based on:

* Use Cases
* Repositories
* Separation of concerns
* Isolated business logic

---

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/adonaycosta/find-a-friend-api.git

# Navigate to the project folder
cd find-a-friend-api

# Install dependencies
npm install

# Run the application
npm run dev
```
